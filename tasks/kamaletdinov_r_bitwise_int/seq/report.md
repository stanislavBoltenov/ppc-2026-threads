## 1. Контекст

Последовательная версия алгоритма поразрядной сортировки служит эталоном для проверки корректности и базой для оценки
ускорения параллельных реализаций. Она реализует устойчивую сортировку целых чисел со знаком.

## 2. Постановка задачи

- **Вход**: Массив целых чисел со знаком (32-бит).
- **Выход**: Отсортированный по возрастанию массив.
- **Ограничения**: Необходимо корректно обрабатывать как положительные, так и отрицательные числа, используя принципы
  поразрядного распределения.

## 3. Базовый алгоритм

Алгоритм реализует метод поразрядной сортировки в десятичной системе счисления с использованием устойчивой сортировки подсчетом.

Так как массив может содержать отрицательные числа, для них принцип сортировки по разрядам имеет краевой случай смены
порядка. Элементы первоначально разделяются на два списка: положительные и отрицательные (сохраняются по модулю).

К каждому из этих подмассивов применяется поразрядная сортировка:

1. **Разделение по разрядам**: Для каждого разряда (от младшего к старшему) выполняется устойчивая сортировка.
2. **Сортировка подсчетом**:
   - Вычисляется гистограмма частот встречаемости цифры в рассматриваемом разряде для каждого числа.
   - Вычисляются префиксные суммы для определения точных смещений размещения каждой цифры в выходном массиве.
   - Значения перераспределяются во временный выходной массив согласно их разряду и префиксным суммам.
3. **Обработка знака**: После сортировки отрицательные элементы инвертируются и записываются обратно в порядке
   убывания в начало исходного массива, за которыми следуют отсортированные положительные элементы.

**Сложность:**

- По времени: $O(d * n)$, где $d$ — количество разрядов (для 32-бит чисел $d \approx 10$), $n$ — размер массива.
- По памяти: $O(n)$ для вспомогательного массива при сортировке подсчетом.

Результат сравнивается с эталонным с помощью `std::ranges::is_sorted` и проверки сохранения всех элементов (сравнение
длин и сумм).
Функциональные тесты покрывают:

- Массивы из одного элемента.
- Уже отсортированные массивы.
- Массивы, отсортированные в обратном порядке.
- Массивы только с отрицательными или только с положительными числами.

void CountingSortByDigit(std::vector<int> &arr, int exp) {
  int n = static_cast<int>(arr.size());
  std::vector<int> output(n);
  std::array<int, 10> count = {};

  for (int i = 0; i < n; i++) {
    count.at((arr[i] / exp) % 10)++;
  }

  for (int i = 1; i < 10; i++) {
    count.at(i) += count.at(i - 1);
  }

  for (int i = n - 1; i >= 0; i--) {
    output[count.at((arr[i] / exp) % 10) - 1] = arr[i];
    count.at((arr[i] / exp) % 10)--;
  }

  arr = output;
}

void BitwiseSort(std::vector<int> &arr) {
  if (arr.size() <= 1) return;
  std::vector<int> neg;
  std::vector<int> pos;

  for (int x : arr) {
    if (x < 0) neg.push_back(-x);
    else pos.push_back(x);
  }

  RadixSortPositive(neg);
  RadixSortPositive(pos);

  std::size_t idx = 0;
  for (int i = static_cast<int>(neg.size()) - 1; i >= 0; i--) {
    arr[idx++] = -neg[i];
  }
  for (int p : pos) {
    arr[idx++] = p;
  }
}
